# wechat

微信收发消息工具

## 功能

1. 获取新消息 - 获取未读的微信消息
2. 发送消息 - 向指定好友发送消息
3. 获取监听列表 - 查看当前正在监听的好友
4. 设置监听列表 - 设置需要自动监听消息的好友

## 用法

### 获取新消息

**获取所有未读消息:**
```bash
python wechat.py getNewData
```

**获取指定好友的未读消息:**
```bash
python wechat.py getNewData "张三"
```

**返回示例:**
```
[张三] 你好 (2024-01-01 12:00:00)
[李四] 在吗 (2024-01-01 12:01:00)
```

---

### 发送消息

**调用:**
```bash
python wechat.py sendMsg "张三" "你好，收到了"
```

**返回:**
```
消息已发送给 张三
```

---

### 获取监听列表

**调用:**
```bash
python wechat.py getMonitorList
```

**返回示例:**
```
当前监听人列表:
  - 张三
  - 李四
```

---

### 设置监听列表

**设置单个好友:**
```bash
python wechat.py setMonitorList "张三"
```

**设置多个好友:**
```bash
python wechat.py setMonitorList "张三" "李四" "王五"
```

**返回:**
```
监听列表已更新: 张三, 李四, 王五
```

---

## 注意事项

- 需要设置监听列表，能能获取最新消息。
